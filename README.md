# 乡村碳足迹 (EcoRural)

一个结合"碳中和 + 健身 + 乡村助农"的移动端 Web App (H5)。

## 技术栈

- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS
- **图标**: Lucide-React
- **语言**: TypeScript

## 功能模块

1. **首页 (Dashboard)** - 展示用户数据、快捷入口和推荐路线
2. **运动/行为记录页 (Activity Tracker)** - 记录运动轨迹和减碳数据
3. **积分集市 (Marketplace)** - 使用绿农币兑换农产品
4. **个人中心 (Profile)** - 查看碳账户、订单和足迹

## 快速预览（无需安装）

**最简单的方式：**
直接双击打开 `index.html` 文件即可在浏览器中查看效果！

这是一个完整的单页 HTML 应用，包含所有功能，无需安装任何依赖或启动服务器。

## 开发（Next.js 版本）

### 方式一：使用一键启动脚本（推荐）

**Windows 用户：**
- 双击运行 `start.bat` 文件
- 或在 PowerShell 中运行 `.\start.ps1`

脚本会自动：
- 检查 Node.js 是否安装
- 检测并安装依赖（如果未安装）
- 启动开发服务器

### 方式二：手动启动

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看应用。
